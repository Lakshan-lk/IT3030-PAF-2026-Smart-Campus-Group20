## § Auth API
| Endpoint | Method | Auth | Notes |
|---|---|---|---|
| /oauth2/authorization/google | GET | None | Spring auto-generates; initiates Google OAuth2 |
| /api/v1/auth/me | GET | Bearer JWT | Returns current user profile |
| /api/v1/auth/users | GET | ADMIN | Query param `?role=TECHNICIAN` for technician list |

**JWT format:** `Authorization: Bearer <token>`
**JWT claims:** `sub` = user email, `userId` = user id (Long), `role` = role string, `exp` = expiry
**Token lifetime:** 24 hours
**OAuth2 success redirect:** `http://localhost:5173/auth/callback?token=<jwt>` (frontend handles storage)

## § WebSocket
- Connect endpoint: `/ws` (SockJS fallback enabled)
- STOMP broker prefix: `/topic`, `/user`
- App destination prefix: `/app`
- Subscribe for notifications: `/user/queue/notifications`
- Push payload: NotificationDTO (same as REST response)
- Auth: send JWT as query param on connect: `/ws?token=<jwt>` — parsed in `HandshakeInterceptor`
